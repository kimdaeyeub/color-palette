# Color Palette

## 프로젝트 동기

프로젝트를 만들다보면 색상의 조합이 굉장히 중요하다고 생각한다. 어떤 색상을 어떻게 조합을 하느냐에 따라 웹페이지의 느낌이 달라질 수 있기 때문이다. 하지만 대부분의 사람들은 색상조합에 어려움을 가질 것 같아서 좋은 색상들을 그룹으로 묵어서 공유하면 좋을것 같다는 생각에 만들게 되었다.

## 스택

- Next.js
- tailwind.css
- typescript
- next-auth
- mongodb
